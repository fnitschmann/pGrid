## Content of this package ##

* Welcome, this is the full package of the pGrid framework. It contains more than just the 
  framework.

* /compressed
  In this folder is a small and compressed version of the pGrid framework. It contains nearly
  the whole /pgrid folder in a single file without any comments.

* /documentation
  This folder contains the offical pGrid-Doc as .pdf file in English and German

* /examples
  In this folder are a few examplex to show what is possible with pGrid

* /license 
  This folder contains information about the license 

* /pgrid
  This folder contains all framwork files. You can directly upload this folder on your webserver etc.

* /tools
  In this folder you can find serveral tools for developing.
