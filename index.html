pGrid | (X)HTML/CSS Framework

Version: 0.2 Release Date: 01.07.2010

ENGLISH

Welcome to the pGrid | (X)HTML/CSS framework. pGrid is a easy framework to write and design multicolumn layouts with less code. The framework gives you a great support and many options to create your layouts really fast and with less HTML and CSS. You have many options to use, given by pGrid, so as HTML5 and CSS3.

(NOTE: In current Version (o.2) it's the aim to work and handle with the new features of HTML5 and CSS3. That features are already supported by many browsers, but not all. If you like to work with classical (X)HTML and CSS2.1/CSS2 look at http://github.com/parcydo/pGrid/downloads for other versions of pGrid.)

  2. Support pGrid is a free and opensource framework. To get support read the PDF files in the folder /doc or look at http://pgridcss.com/doc

  3. Changelog Every new version comes with many changes and bugfixes. To get knowlege about all this read the CHANGELOG file.

  4. Structure

  • /compressed Here you can find the compressed and minimum version of the pGrid Framework. Please read the doumentation before usage.

  • /doc This is the full documentation of the pGrid in German and English. It comes as a PDF file.

  • /examples Show and give you some examples what's possible with pGrid. All this layouts are free in usage and under the MIT License, so you can use them free and for your own projects if you like.

  • /src This is the framework folder. Here you can find all important files. You can directly upload this folder on your webserver if you like.

  • /tools These are some "extra tools" wich help you developing websites.

DEUTSCH

Willkommen zum pGrid | (X)HTML/CSS Framework. pGrid ist ein einfaches Framwork um vielspaltige Layouts mit wenig Code zu schreiben und gestallten. Das Framwork bietet viele Optionen um Layouts schnell mit wenig HTML und CSS zu entwickeln. Mit den von pGrid gegebenen Optionen hat man viele Möglichkeiten, wie zum Beispiel HTML5 und CSS3.

(HINWEIS: Die aktuelle Version (0.2) ist darauf ausgelegt mit HTML5 und CSS3 zu arbeiten. Diese Features werden schon von vielen Browsern unterstützt, jedoch noch nicht von allen. Wenn Sie mit klassichen (X)HTML und CSS2.1/CSS2 arbeiten wollen schauen unter http://github.com/parcydo/pGrid/downloads nach anderen pGrid Versionen.)
